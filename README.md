# Comic Tracker

An AI-powered manga/manhwa/manhua tracker that helps you find comics based on descriptions, characters, and settings.

## Features (Planned)

- **Agent A**: Research bot that automatically gathers comic information from the web
- **Agent B**: Query bot that helps users find comics based on fuzzy descriptions
- Simple search interface with genre/type filters
- Local database of comic information

## Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: Node.js + Express
- **Database**: JSON (may upgrade to SQLite later)
- **AI**: OpenAI GPT API

## Setup

1. Install dependencies:
   ```bash
   cd backend
   npm install
   ```

2. Start the server:
   ```bash
   npm start
   ```

3. Open your browser to `http://localhost:3000`
